[ 01 ] APA ITU KAMERA DI SFML?

Dalam SFML, kamera disebut sf::View. Kamera ini adalah area pandang (viewport) yang menentukan bagian mana dari dunia game yang terlihat di layar.

Dengan menggerakkan kamera, kita bisa membuat efek seperti game RPG, sandbox, ataupun open-world 2D.

Untuk membuat gerakan player akurat, arah harus dinormalisasi:

\[ \text{arah} = (x, y) \] \[ \text{panjang} = \sqrt{x^2 + y^2} \] \[ \text{arah\_normal} = \frac{(x, y)}{\text{panjang}} \]
[ INFO ]

Kamera TIDAK digambar! Tetapi kamera mengatur posisi dunia game yang terlihat oleh pemain.

[ 02 ] CARA MEMBUAT KAMERA DI SFML

[ C++ ] Membuat Kamera
sf::View kamera(
    sf::Vector2f(0.f, 0.f),              // posisi pusat kamera
    sf::Vector2f(panjangLayar, tinggiLayar)  // ukuran kamera
);
jendela.setView(kamera);

Parameter pertama adalah center, dan yang kedua adalah size. Kamera dapat punya ukuran berbeda dari ukuran window (berguna untuk zoom in/out).

[ 03 ] MEMBUAT KAMERA MENGIKUTI PLAYER

Paling mudah: cukup set center kamera menjadi posisi player setiap frame.

[ C++ ] Kamera Follow Player
kamera.setCenter(lingkaran.getPosition());
jendela.setView(kamera);

Dengan satu baris ini, kamera akan selalu berada tepat di tengah player.

[ CATATAN ]
Kamera bekerja seperti menerjemahkan dunia game, bukan memindahkan player!

[ 04 ] SMOOTH CAMERA (LERP)

Kamera yang langsung "snap" ke player terkadang terlihat kasar. Kita bisa membuat kamera bergerak halus menggunakan linear interpolation (LERP).

[ C++ ] Smooth Camera Lerp
sf::Vector2f target = lingkaran.getPosition();
sf::Vector2f current = kamera.getCenter();

// faktor smoothing (0.0 - 1.0)
float smooth = 0.1f;

sf::Vector2f newPos = current + (target - current) * smooth;

kamera.setCenter(newPos);

Hasilnya adalah kamera yang mengikuti player tetapi tetap halus seperti game RPG modern.

[ 05 ] MEMBATASI KAMERA AGAR TIDAK KELUAR MAP

Agar kamera tidak menunjukkan area hitam di luar map, kita dapat membatasi center kamera.

[ C++ ] Membatasi Kamera
float halfX = kamera.getSize().x / 2.f;
float halfY = kamera.getSize().y / 2.f;

sf::Vector2f pos = kamera.getCenter();

// batas kiri & kanan
pos.x = std::clamp(pos.x, halfX, mapWidth - halfX);

// batas atas & bawah
pos.y = std::clamp(pos.y, halfY, mapHeight - halfY);

kamera.setCenter(pos);

Ini sangat penting jika map kamu besar (open world).

[ 06 ] CONTOH KODE KAMERA FOLLOW PLAYER (LENGKAP)

Berikut kode yang paling umum dipakai untuk kamera yang mengikuti player di SFML 3.0:

[ C++ ] Full Example
sf::CircleShape player(40.f);
player.setOrigin(player.getLocalBounds().getCenter());
player.setPosition(500.f, 500.f);

sf::View kamera(
    sf::Vector2f(0.f, 0.f),
    sf::Vector2f(panjangLayar, tinggiLayar)
);

...

// update camera setiap frame
kamera.setCenter(player.getPosition());
jendela.setView(kamera);
