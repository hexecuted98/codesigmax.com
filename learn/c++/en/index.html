Let's Learn C++ Programming

Introduction to C++

C++ is a powerful general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language.

Your First C++ Program

Here's a simple "Hello World" program in C++:

hello_world.cpp 
#include <iostream>

int main() {
    // This is a simple C++ program
    std::cout << "Hello, World!" << std::endl;
    
    // Variables and basic operations
    int number = 42;
    double pi = 3.14159;
    std::string message = "Learning C++ is fun!";
    
    std::cout << "The number is: " << number << std::endl;
    std::cout << "PI is approximately: " << pi << std::endl;
    std::cout << message << std::endl;
    
    return 0;
}

Key components of this program:

  • #include <iostream> - Header for input/output operations
  • int main() - The main function where execution begins
  • std::cout - Standard output stream
  • std::endl - End line manipulator
  • return 0 - Indicates successful program execution
C++ Basics

Variables and Data Types

variables.cpp 
#include <iostream>
#include <string>

int main() {
    // Integer types
    int age = 25;
    short smallNumber = 32767;
    long bigNumber = 2147483647;
    
    // Floating point types
    float price = 19.99f;
    double pi = 3.1415926535;
    
    // Character types
    char grade = 'A';
    bool isCppFun = true;
    
    // String type
    std::string name = "Alice";
    
    // Output all variables
    std::cout << "Age: " << age << std::endl;
    std::cout << "Small number: " << smallNumber << std::endl;
    std::cout << "Price: " << price << std::endl;
    std::cout << "PI: " << pi << std::endl;
    std::cout << "Grade: " << grade << std::endl;
    std::cout << "Is C++ fun? " << std::boolalpha << isCppFun << std::endl;
    std::cout << "Name: " << name << std::endl;
    
    return 0;
}
Object-Oriented Programming in C++

Classes and Objects

class_example.cpp 
#include <iostream>
#include <string>

// Define a class
class Car {
private:
    std::string brand;
    std::string model;
    int year;
    
public:
    // Constructor
    Car(std::string b, std::string m, int y) : brand(b), model(m), year(y) {}
    
    // Method to display car information
    void displayInfo() {
        std::cout << "Brand: " << brand << std::endl;
        std::cout << "Model: " << model << std::endl;
        std::cout << "Year: " << year << std::endl;
    }
    
    // Setter method
    void setYear(int newYear) {
        year = newYear;
    }
};

int main() {
    // Create an object of Car
    Car myCar("Toyota", "Corolla", 2020);
    
    // Call methods
    myCar.displayInfo();
    
    // Modify property using 
    myCar.setYear(2022);
    std::cout << "\nAfter update:" << std::endl;
    myCar.displayInfo();
    
    return 0;
}
